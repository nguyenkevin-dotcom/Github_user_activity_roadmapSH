# Github_user_activity_roadmapSH

In this project, I will build a simple command line interface (CLI) to fetch the recent activity of a GitHub user (nguyenkevin-dotcom) and display it in the terminal. This project will help me practice my programming skills, including working with APIs, handling JSON data, and building a simple CLI application.

## Description

Shows the recent activity of user in github.com

## Instructions

Type on terminal `pip install -e .`, the last lines of output should be:

```

  Attempting uninstall: github_user_activity_package_nguyenkevin-dotcom
    Found existing installation: github_user_activity_package_nguyenkevin-dotcom 0.0.1
    Uninstalling github_user_activity_package_nguyenkevin-dotcom-0.0.1:
      Successfully uninstalled github_user_activity_package_nguyenkevin-dotcom-0.0.1
Successfully installed github_user_activity_package_nguyenkevin-dotcom-0.0.1
```

Then type on terminal type `github-activity <github-username>`
